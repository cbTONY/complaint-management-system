# Complaint Management System

## Project Overview
The Complaint Management System is a web-based application developed to help users submit complaints and allow administrators to manage and resolve them efficiently. The system improves transparency and simplifies the complaint handling process.

## Features
- User registration and login
- Submit complaints
- Track complaint status
- Admin dashboard to manage complaints
- Update complaint status

## Technologies Used
- Node.js
- Express.js
- MySQL
- HTML
- CSS
- JavaScript

## Project Structure
- server.js – Main application server
- db.js – Database connection
- views/ – Frontend pages
- public/ – CSS and static files
- package.json – Project dependencies

## How to Run the Project

1. Install Node.js
2. Install project dependencies
   npm install

3. Start the server
   node server.js

4. Open the browser and go to
   http://localhost:3000
